# Practica-3-Isabel-Parra-y-Mariana-Montoya-

This project implements a parser for Forsyth–Edwards Notation (FEN) in Java.
It reads a FEN string, validates its syntax, and displays the chessboard using Unicode symbols.
If the FEN is invalid, it prints an error message describing the issue.

Files:
Main.java – Entry point, reads FEN input.
Parser.java – Splits and validates FEN fields.
Validator.java – Checks syntax and logic.
Board.java – Displays the chessboard.
Piece.java – Represents chess pieces.
